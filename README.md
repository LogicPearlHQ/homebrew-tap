# LogicPearl Homebrew Tap

Homebrew tap for LogicPearl release formulae.

After a LogicPearl release is published, install with:

```bash
brew install LogicPearlHQ/tap/logicpearl
```

The `Formula/logicpearl.rb` file is generated from LogicPearl GitHub Release bundle checksums by the main repository release workflow.
